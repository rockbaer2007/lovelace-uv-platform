# Architecture Principles

P-001 Public API first
P-002 Internal implementation hidden
P-003 Foundation contains only generic concepts
P-004 Renderer independent architecture
P-005 Tree-based layouts
P-006 Design tokens instead of CSS values
P-007 Documentation before implementation
P-008 No breaking public API without ADR
P-009 Small, verifiable commits
P-010 Every subsystem follows Manager / Context / Metadata pattern
