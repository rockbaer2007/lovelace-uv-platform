# Release Notes

## Genesis 0.1.x

Genesis establishes the architectural foundation of the Lovelace UV Platform.

Highlights

- Modular platform architecture
- Generic Registry
- Shared Contracts
- Event Bus
- Plugin System
- Provider API
- Resource Model
- Architecture documentation
- Public/Internal API separation

The next release train (Atlas 0.2.x) focuses on runtime services,
provider management and extensibility.
